## Setting Meta Tags in Nuxt 3

You can manage meta tags using the `useHead()` composable:

```ts
<script setup>
useHead({
  title: 'Nuxt 3 SEO Best Practices',
  meta: [
    { name: 'description', content: 'Learn how to optimize SEO in your Nuxt 3 app' },
    { name: 'og:title', content: 'Nuxt 3 SEO Guide' },
    { name: 'og:type', content: 'website' },
    { name: 'twitter:card', content: 'summary' }
  ]
})
</script>
